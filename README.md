**FoodShare** is a web-based platform that connects donors with food banks and organizations to reduce food waste and support those in need. Donors can easily offer food, track donations, communicate with organizers via chat, and manage their profile settings.



## Features

### Donor Side
- **Dashboard:** Overview of donations and quick actions.  
- **Food Donation Form:** Submit details about food to donate.  
- **Pickup Schedule:** Track upcoming pickup schedules.  
- **My Donations:** View status of donations (Pending, Approved, Delivered).  
- **Chat Module:** Real-time messaging with organizers/donors.  
- **Settings Page:** Update profile, change password, manage notifications.  

### Notifications
- Users receive alerts for donation approval, pickup schedules, and messages.
